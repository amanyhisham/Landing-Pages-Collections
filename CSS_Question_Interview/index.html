CSS Interview Guide

Complete reference for CSS interview questions

Box Model (8 questions)

1

What is the CSS Box Model?

Structure of every element: content, padding, border, margin.

هيكل كل عنصر في الصفحة بيتكون من: المحتوى، المسافة الداخلية، الإطار، والمسافة الخارجية

2

Explain the difference between content, padding, border, and margin

Content: actual text/image inside • Padding: space between content and border • Border: frame around element • Margin: space outside element

Content: النص أو الصورة داخل الصندوق • Padding: المسافة داخل البوكس بين المحتوى والحد • Border: الإطار المحيط بالعنصر • Margin: المسافة خارج العنصر بينه وبين العناصر الأخرى

3

What does box-sizing: border-box do?

Makes width and height include padding and border (doesn't increase total size).

يجعل العرض والارتفاع يشملان الـ padding والـ border، يعني ما يزودش الحجم الكلي

4

What's the default box-sizing value?

content-box (padding and border add to the total size).

content-box يعني الـ padding والـ border بيتضافوا فوق الحجم

5

What is margin collapsing?

When vertical margins meet (top + bottom), they take only the larger value instead of adding together.

لما مارجن رأسيين يتقابلوا، بياخدوا قيمة الأكبر فقط بدل ما يتجمعوا

6

Difference between border and outline

Border is inside the box and affects size • Outline is outside and doesn't affect size.

border داخل البوكس وبتأثر على الحجم، outline بره البوكس ومش بتأثر على الحجم

7

Can margins be negative?

Yes, negative margin moves the element in that direction.

أيوه، margin السالب بيخلّي العنصر يتحرك ناحية الاتجاه ده

8

Does padding accept negative values?

No, padding cannot be negative.

لأ، الـ padding ما ينفعش يكون سالب

CSS Overflow (6 questions)

1

What does the overflow property do in CSS?

Controls what happens when content overflows an element's box.

بتتحكم في اللي بيحصل لما المحتوى يكون أكبر من المساحة المحددة للعنصر

2

What are the possible values for overflow?

visible, hidden, scroll, and auto.

visible: المحتوى بيطلع برا • hidden: بيخفي الزيادة • scroll: شريط تمرير دايماً • auto: شريط تمرير لما يحتاج بس

3

What's the default value of overflow?

The default value is visible.

القيمة الافتراضية visible، يعني المحتوى هيطلع برا عادي

4

What's the difference between overflow: scroll and overflow: auto?

scroll: always shows scrollbars • auto: shows scrollbars only when needed.

scroll بيظهر الـ scrollbar دايماً، auto بيظهره بس لما المحتوى يكون كبير فعلاً

5

Can you control horizontal and vertical overflow separately?

Yes, using overflow-x and overflow-y.

أيوه، overflow-x للأفقي وoverflow-y للرأسي

6

What happens if you set overflow: hidden?

The overflowing content will be clipped and not visible.

أي حاجة تطلع برا حدود العنصر هتتقصّ ومش هتظهر خالص

Display Property (7 questions)

1

What is display: inline?

Element stays on same line with others, can't set width/height, takes only content space.

العنصر بيكون على نفس السطر مع غيره، ما ينفعش نديه width أو height، بياخد مساحة على قد المحتوى

2

What is display: block?

Takes full line alone, can set width and height.

بياخد صف كامل لوحده، نقدر نديه width وheight

3

What is display: inline-block?

Mix of both: stays on same line like inline, but can set width/height like block.

خليط: بيكون على نفس السطر زي inline، لكن نقدر نديه width وheight زي block

4

What is display: none?

Element completely disappears from page as if it doesn't exist (takes no space).

العنصر بيختفي تماماً من الصفحة وكأنه مش موجود، ما بياخدش مساحة

5

Difference between display: none and visibility: hidden?

none: removes element completely • hidden: hides element but keeps its space.

none بيخفي العنصر وبيشيل مكانه، hidden بيخفيه لكن بيحافظ على مكانه

6

What is display: flex?

Makes parent a flex container, arranges children in one dimension (row or column).

بيخلي العنصر الأب flex container، ينظم العناصر في بعد واحد (صف أو عمود)

7

What is display: grid?

Controls layout in two dimensions (rows and columns), ideal for complex layouts.

تتحكم في بعدين (صفوف وأعمدة)، مثالية للتصميمات المعقدة

Position Property (8 questions)

1

What is position: static?

Default positioning, element follows normal document flow.

الوضع الافتراضي، العنصر بيتبع التدفق الطبيعي للصفحة

2

What is position: relative?

Element moves relative to its original position, keeps space in document.

العنصر بيتحرك بالنسبة لمكانه الأصلي، بس بيحتفظ بمكانه في الصفحة

3

What is position: absolute?

Element positioned relative to nearest positioned parent (or body), removed from flow.

العنصر بيتحرك بالنسبة لأقرب أب عنده position غير static، بيخرج من التدفق الطبيعي

4

What is position: fixed?

Element fixed relative to viewport, stays in place when scrolling.

العنصر ثابت بالنسبة للشاشة، بيفضل في مكانه حتى لو سكرولت

5

What is position: sticky?

Mix of relative and fixed: scrolls normally until threshold, then becomes fixed.

خليط: بيتحرك عادي لحد ما يوصل لنقطة معينة، بعدها بيثبت زي fixed

6

Difference between absolute and relative?

relative: keeps space in flow • absolute: removed from flow, positioned relative to parent.

relative بيحافظ على مكانه في الصفحة، absolute بيخرج من التدفق ويتحرك بالنسبة للأب

7

How to center element with position: absolute?

Use top: 50%; left: 50%; transform: translate(-50%, -50%);

استخدم top: 50%; left: 50%; transform: translate(-50%, -50%); عشان يبقى في النص بالضبط

8

Why doesn't sticky work sometimes?

Must specify top/bottom value, parent can't have overflow: hidden or auto.

لازم تحدد top أو bottom، والأب ما يكونش عليه overflow: hidden

Z-Index & Stacking (3 questions)

1

When does z-index not work?

When element has position: static (default).

لما يكون العنصر position: static، لازم يكون relative أو absolute أو fixed

2

Difference between positive and negative z-index?

Positive: brings element forward • Negative: pushes element backward.

الموجب بيخلي العنصر فوق، السالب بيخليه تحت

3

What if two elements have same z-index?

The one that comes later in HTML appears on top.

اللي جاي بعد في الكود بيظهر فوق

Background Properties (1 question)

1

What is background-attachment?

Controls if background moves with page or stays fixed. Values: scroll (moves) or fixed (stays).

بتحدد هل الخلفية تتحرك مع الصفحة (scroll) ولا ثابتة (fixed)

Flexbox (8 questions)

1

What is the purpose of Flexbox?

Arranges elements in a container flexibly and easily, controlling direction and alignment horizontally or vertically without float or position.

بيُستخدم لترتيب العناصر داخل الحاوية بطريقة مرنة وسهلة، بتوزع المسافات وتتحكم في الاتجاه أفقي أو رأسي من غير float

2

Difference between justify-content and align-items?

justify-content: controls horizontal distribution (main axis) • align-items: controls vertical distribution (cross axis).

justify-content بتتحكم أفقياً، align-items بتتحكم رأسياً

3

Difference between align-items and align-content?

align-items: aligns elements within one line • align-content: aligns all rows when using flex-wrap.

align-items بتتحكم في محاذاة العناصر داخل سطر واحد، align-content بتتحكم في كل الصفوف

4

What does flex-direction do?

Determines the direction of elements: row (horizontal) or column (vertical).

بتحدد اتجاه العناصر: صف (row) أو عمود (column)

5

Difference between space-between, space-around, and space-evenly?

space-between: first/last touch edges, space between only • space-around: equal space around each • space-evenly: all spaces equal.

space-between: أول وآخر ملتصقين بالأطراف • space-around: مسافات حول كل عنصر • space-evenly: كل المسافات متساوية

6

What does flex-wrap do?

Makes elements wrap to new line when space isn't enough instead of staying in one line.

بتخلي العناصر تنزل لسطر جديد لو المساحة مش كفاية

7

What does the order property do?

Changes element order within flex container without changing HTML code.

بتغير ترتيب العنصر داخل flex من غير ما تغير الكود

8

What happens with flex-wrap: nowrap?

All elements stay in one line even if there's not enough space, might overflow container.

العناصر كلها هتفضل في سطر واحد حتى لو المساحة مش كفاية

CSS Grid (5 questions)

1

Difference between Flexbox and Grid?

Flexbox: one-dimensional layout (row or column) • Grid: two-dimensional layout (rows + columns together).

Flexbox بُعد واحد (سطر أو عمود)، Grid بُعدين (صفوف + أعمدة مع بعض)

2

What do grid-template-columns and grid-template-rows do?

grid-template-columns: defines number and size of columns • grid-template-rows: defines number and size of rows.

grid-template-columns تحدد عدد الأعمدة وحجمهم، grid-template-rows تحدد الصفوف

3

Difference between gap, row-gap, and column-gap?

gap: space between rows and columns together • row-gap: space between rows only • column-gap: space between columns only.

gap للصفوف والأعمدة مع بعض، row-gap للصفوف بس، column-gap للأعمدة بس

4

What do justify-items and align-items do in Grid?

justify-items: horizontal alignment within cells • align-items: vertical alignment within cells.

justify-items محاذاة أفقية داخل الخلية، align-items محاذاة رأسية

5

Difference between justify-content and align-content in Grid?

justify-content: aligns all columns horizontally • align-content: aligns all rows vertically.

justify-content محاذاة كل الأعمدة أفقياً، align-content محاذاة كل الصفوف رأسياً

Transitions & Animations (4 questions)

1

Difference between CSS Transitions and Animations?

Transition: smooth change on specific event (hover, focus) • Animation: continuous/programmed effects using @keyframes.

Transition تغيير سلس عند حدث معين، Animation تأثيرات مستمرة باستخدام @keyframes

2

What are the components of transition?

property (what changes), duration (how long), timing-function (movement shape), delay (wait before start).

property: الخاصية • duration: المدة • timing-function: شكل الحركة • delay: التأخير

3

What are the components of animation?

@keyframes (defines movement), animation-name, animation-duration, animation-iteration-count (repetitions).

@keyframes تحدد الحركة، animation-name الاسم، animation-duration المدة، iteration-count عدد التكرار

4

How to make animation repeat infinitely?

Use animation-iteration-count: infinite;

استخدم animation-iteration-count: infinite; عشان تتكرر بلا حدود

Pseudo-classes & Pseudo-elements (4 questions)

1

Difference between pseudo-class and pseudo-element?

Pseudo-class: styles element in specific state (:hover, :focus) • Pseudo-element: styles specific part (::before, ::after).

Pseudo-class بتستهدف حالة معينة للعنصر، Pseudo-element بتستهدف جزء معين منه

2

Difference between :hover, :active, and :focus?

:hover (mouse over) • :active (while clicking) • :focus (when selected/tabbed to).

:hover لما الماوس فوق العنصر • :active وقت الضغط • :focus لما العنصر يكون محدد

3

How to add content before element with CSS?

Use ::before with content property.

استخدم ::before مع خاصية content

4

When to use :first-child and :last-child?

To target first or last element in a group for special styling.

لاستهداف أول أو آخر عنصر في مجموعة عشان تديه تنسيق خاص
